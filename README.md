## 🌟 Core Functionalities

### 🤖 AI Fitment Diagnosis
Our AI Fitment Expert analyzes complex compatibility questions (e.g., "Can I fit 5x114.3 wheels on a 2018 Civic?") by cross-referencing real-time technical datasets with structural safety parameters.

### 🔍 Comprehensive Database
- **Vehicle Search**: Quick lookup for PCD, Hub Bore, Offset, and Thread pitch by make, model, and year.
- **Reverse PCD Lookup**: Find all vehicles that share a specific bolt pattern (e.g., 5x112, 5x120).
- **Expert Guides**: Deep dives into wheel offset, hub-centric vs. lug-centric mounting, and safe bolt pattern conversion.

## 🛠️ Technical Implementation
- **Cloudflare Workers Backend**: Leveraging high-performance edge computing for AI diagnostics via `bolt.rsslite.workers.dev`.
- **Dynamic Search Indexing**: Client-side filtering of massive vehicle datasets for instantaneous search results.
- **Schema-Rich Architecture**: Optimized with JSON-LD for enhanced search engine visibility and rich automotive snippets.
